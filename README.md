# 中国大学视觉形象识别系统网址集合  
## Visual Identity System of Chinese Universities

视觉形象识别系统(Visual Identity System of Chinese University, VIS) 是由代表学校视觉形象的识别元素及其组合变化构成的系统，适用于学校及各二级单位办公会务、形象宣传、文体活动等方面。

本项目收集了高校视觉形象识别系统网址，方便各校师生下载所需矢量素材。
按教育部公布的[全国高等学校名单](http://www.moe.gov.cn/jyb_xxgk/s5743/s5744/A03/202306/t20230619_1064976.html)学校识别码从小到大排序
